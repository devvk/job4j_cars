# Job4j Cars

## О проекте

Веб-приложение для размещения объявлений о продаже автомобилей.

Пользователь может зарегистрироваться, войти в систему, создать объявление о продаже машины, добавить описание,
характеристики автомобиля и фотографии. Автор объявления может менять статус объявления на «продано».

## Функционал

- регистрация пользователя;
- авторизация пользователя;
- просмотр списка объявлений;
- просмотр деталей объявления;
- создание нового объявления;
- добавление фотографий к объявлению;
- изменение статуса объявления;
- отображение автора объявления;
- фильтрация объявлений:
    - за последний день;
    - с фотографиями;
    - по марке автомобиля.
